| Role Name             | الوصف                                                                           |
| --------------------- | ------------------------------------------------------------------------------- |
| **admin**             | المدير العام للنظام، ليه صلاحيات كاملة (إضافة/تعديل/حذف أي شيء)                 |
| **store\_manager**    | مدير المخزن، مسؤول عن الإشراف على حركة البضائع وإدارة المخزون                   |
| **inventory\_clerk**  | موظف الجرد، مسؤول عن تسجيل عمليات الإدخال والإخراج للمخزن                       |
| **purchasing\_agent** | مسؤول المشتريات، يتابع طلبات الشراء من الموردين                                 |
| **sales\_rep**        | موظف المبيعات، يطلع على الكميات المتاحة ويسجل عمليات البيع                      |
| **accountant**        | المحاسب، يتابع الفواتير والحسابات والتكاليف المرتبطة بالمخزون                   |
| **auditor**           | المدقق، يطلع فقط على تقارير النظام بدون صلاحية تعديل                            |
| **supplier**          | مورد خارجي، قد يكون له لوحة محدودة لمتابعة الطلبات (لو النظام فيه بوابة موردين) |
| **customer**          | عميل نهائي، فقط يطلع على حالته/طلباته (لو فيه طلبات من العملاء مثلاً)           |
| **data\_entry**       | موظف إدخال بيانات، فقط يدخل بيانات المنتجات أو الفواتير                         |
| **viewer**            | مستخدم يقدر يشوف فقط تقارير أو بيانات بدون صلاحيات تعديل                        |
